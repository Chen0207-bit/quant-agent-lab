# Completion Reports

This directory stores one Markdown completion report for each completed Codex project task.

Rules:

- Write reports only under `/home/fc/projects/quant-agent-lab/docs/completion_reports/`.
- Use `Asia/Shanghai` time for filenames and report timestamps.
- Use filename format `YYYY-MM-DD_HHMM_<short_task_slug>.md`.
- If a filename already exists for the same minute and slug, append `-2`, `-3`, and so on.
- Do not write these reports in the Windows workspace.
- Do not create a report for casual chat or non-project Q&A.

Template:

```markdown
# Task Completion - YYYY-MM-DD HH:MM

## Goal

## Completed

## Verification

## Changed Files

## Notes / Next Steps
```
