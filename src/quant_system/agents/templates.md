# Agent Task Template

## Goal

State the bounded research or review goal.

## Inputs

List source documents, data snapshots, configs, and prior reports.

## Output

Return a one-page conclusion, comparison table, risks, and executable recommendation.

## Acceptance

The result must be reproducible and must not modify production risk rules, production configs, or execution permissions.

## Forbidden

- Direct order placement.
- Risk rule override.
- Production parameter mutation without human approval.
- Real-time trading-loop control.
